✏️ Content Update Guide

How to update banners, testimonials, contact details, and other content on the Fintex Global Group website. No coding required for most changes.

📢 1. How to Update Banners (Promotional Slides)

The homepage shows a rotating banner slider with up to 4 promotional slides. Each slide has: an eyebrow label, headline, subtext, two buttons, and a background colour.

1Open index.html in a text editor (e.g. VS Code, Notepad++)

Find the section that starts with:

<!-- BANNER SLIDER -->

Each slide is a <div class="banner-slide"> block. You will see 4 of these.

2Edit the text inside each slide

Each slide contains these parts you can change:

<!-- BANNER SLIDE — edit freely -->
<span class="banner-eyebrow" style="color:#E8B800;">
  🇦🇺 Australia — New Clients     ← CHANGE THIS LABEL
</span>
<h2 class="banner-title">
  Free BAS Review for New AU Clients   ← CHANGE THIS HEADLINE
</h2>
<p class="banner-sub">
  First-time clients get a complimentary...  ← CHANGE THIS TEXT
</p>
<a href="contact.html?service=australia-trial" class="btn btn-au">
  Claim Free Review →   ← CHANGE BUTTON TEXT
</a>
3Add a new banner slide

Copy this template and paste it inside the <div class="banner-track">:

<div class="banner-slide">
  <div class="banner-bg" style="background:linear-gradient(135deg,rgba(0,180,216,.12),rgba(11,24,41,1));"></div>
  <div class="banner-slide-inner">
    <span class="banner-eyebrow" style="background:rgba(255,255,255,.06);border:1px solid rgba(255,255,255,.12);color:#00B4D8;">
      YOUR LABEL HERE
    </span>
    <h2 class="banner-title">Your Headline Here</h2>
    <p class="banner-sub" style="color:rgba(235,244,250,.75);">
      Your promotional message goes here. Keep it to 2–3 sentences.
    </p>
    <div class="banner-btns">
      <a href="contact.html" class="btn btn-primary">Primary Button →</a>
      <a href="services.html" class="btn btn-outline">Secondary Button</a>
    </div>
  </div>
</div>

Then add a corresponding dot button inside <div class="banner-dots">:

<button class="banner-dot" onclick="goSlide(4)" aria-label="Slide 5"></button>

And update the total variable in the slider script from 4 to 5.

4Remove a banner slide

Delete the entire <div class="banner-slide">...</div> block and its matching dot button. Update the total variable accordingly.

5Change the auto-rotate speed

Find setInterval(function(){goSlide(cur+1);}, 5500) in the banner script. Change 5500 to your desired milliseconds (e.g. 4000 = 4 seconds, 8000 = 8 seconds).

💡 Button colour options: btn-au (gold), btn-in (saffron), btn-uae (cyan), btn-primary (blue gradient), btn-outline (transparent/border).

⭐ 2. How to Update Testimonials

Testimonials appear on the homepage below the services section. Each card has a star rating, quote, client name, role, initials and region flag.

1Open index.html and find the testimonials section
<!-- TESTIMONIALS -->
<section class="testimonials-section">
2Edit an existing testimonial card
<div class="testi-card reveal">
  <div class="testi-stars">⭐⭐⭐⭐⭐</div>
  <div class="testi-text">"Your client quote here..."</div>
  <div class="testi-author">
    <div class="testi-ava" style="background:rgba(232,184,0,.1);color:#E8B800;">AB</div>
    <div>
      <div class="testi-name">Client Name</div>
      <div class="testi-role">Role · City, Country</div>
    </div>
  </div>
  <div class="testi-region">🇦🇺</div>
</div>

Change the initials AB, name, role, quote and flag emoji. Colour options: #E8B800 (AU gold), #E07B00 (India saffron), #00B4D8 (UAE cyan).

📞 3. How to Update Contact Details

1Replace all placeholder contacts at once

Use Find & Replace (Ctrl+H in VS Code) across ALL files:

  • REPLACE_AU_WHATSAPP → Arun's number, e.g. 61412345678
  • REPLACE_AU_EMAIL → Arun's email, e.g. arun@tensquare.com.au
  • REPLACE_INDIA_WHATSAPP → India number, e.g. 919876543210
  • REPLACE_INDIA_EMAIL → India email, e.g. india@fintexglobal.com
💡 The UAE WhatsApp (971557282794) and email (info@fintexglobal.com) are already set.

👥 4. How to Add / Update Team Members

Open team.html. Each team card follows this structure. Copy an existing card and update the fields:

  • Change the initials in .tc-ava
  • Update name, title, and qualifications badges
  • Update the info cells (Based in, Specialises in, etc.)
  • Update LinkedIn URL and WhatsApp number

Replace a "Coming Soon" placeholder by deleting the <div class="coming-soon"> block and pasting in a new team card.

📝 5. How to Add a Blog Article

Open blog.html. Find the <div class="blog-grid"> section. Copy this template and paste it at the top:

<a href="contact.html" class="blog-card reveal" data-cat="au">
  <div class="blog-card-stripe" style="background:var(--au-gold);"></div>
  <div class="blog-card-body">
    <div class="blog-card-tag" style="color:var(--au-gold);">🇦🇺 AUSTRALIA</div>
    <div class="blog-card-title">Your Article Title Here</div>
    <div class="blog-card-desc">Brief description of the article — 2 sentences max.</div>
    <span class="blog-read-more" style="color:var(--au-gold);">Read Article →</span>
  </div>
</a>

Set data-cat to: au / in / uae / outsourcing / security to enable the filter.

🎨 6. How to Change Brand Colours

Open shared.css. All colours are CSS variables at the top of the file:

:root{
  --navy:      #0B1829;    /* Page background */
  --cyan:      #00B4D8;    /* Primary accent */
  --au-gold:   #E8B800;    /* Australia colour */
  --in-saffron:#E07B00;    /* India colour */
  --uae-cyan:  #00B4D8;    /* UAE colour */
}

Change any hex value to update that colour globally across the entire site.

🔐 7. How to Update Portal Credentials

  1. Go to portal.html and log in as Admin
  2. Click 🔑 Credentials in the admin menu
  3. Use the Reset button to change any existing password
  4. Use the Add form to create new client or admin accounts
⚠️ Change all demo passwords before publishing. Default credentials are stored in browser localStorage — replace with a proper backend (Supabase/Firebase) for production use.

🚀 8. Publishing / Deploying Updates

1Edit your files locally

Make your changes in a text editor. Save all modified files.

2Re-deploy to Netlify

Option A — Drag & Drop: Go to app.netlify.com → your site → Deploys tab → drag the entire folder onto the deploy dropzone.

Option B — Git: If connected to GitHub, just push your changes and Netlify auto-deploys.

3Verify after deploy

Visit your live domain. Hard refresh (Ctrl+Shift+R) to clear browser cache. Check the banner appears, links work, and the portal login functions correctly.
